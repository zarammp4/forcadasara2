# 🎮 Jogo da Forca em PHP

Um joguinho da forca simples desenvolvido em PHP com sistema de login e sorteio aleatório de palavras.

## 🚀 Funcionalidades

- 🔐 Sistema de login simples
- 🎲 Sorteio aleatório de palavras
- ✅ Validação de letras digitadas
- 🎨 Interface com Bootstrap 5
- 📱 Design responsivo
